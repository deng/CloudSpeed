# CloudSpeed

Upload file  into the filecoin network and share it.

## Configuration

```
vi Configs/appsettings.json
```

## Init Database

```
cd CloudSpeed.Migrations
dotnet run
```

## Usage

```
cd CloudSpeed.Web
dotnet run
```
